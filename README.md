# Windward Aerodynamics

A Minecraft mod that overhauls Sable's aerodynamics to be less arcady and more realism-based using Simulated Aerofoil Data.

Instead of relying on basic lift and drag values, blocks that generate lift are now broken up into spanwise sections. By applying realistic aerofoil calculations that factor in the Angle of Attack, crafts behave much more realistically in the air, allowing planes to handle significantly better.

*Note: Global wind mechanics are not implemented yet. Once wind is added, you will also be able to build functional Rigid Wing sailing ships.*

## Features
- Replaces Sable's arcade-style physics with realistic Simulated Aerofoil Data.
- Force multipliers for fine-tuning aerodynamics.
- Visual debug particles to see lift (orange) and chord normals (blue).
